## Amount units

The `amount` field is always a plain decimal number in the currency's standard unit — the same unit you'd quote a price in, not the smallest/lowest unit (kobo, cents, pence, etc.). This is uniform across every currency listed above:

| To charge | `amount` |
|---|---|
| ₦50.00 (NGN) | `amount: 50` |
| $50.00 (USD) | `amount: 50` |
| £50.00 (GBP) | `amount: 50` |

Decimals are supported where the currency has them — e.g. `amount: 49.99` charges $49.99.

<Note>
  Adding a new currency requires an SDK update, since `Currency` is a fixed TypeScript union. If you need a currency that isn't listed here, contact Reeple.
</Note>
