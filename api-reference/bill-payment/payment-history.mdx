Retrieve a list of all your bill payments.

## Endpoint

```
GET https://api-v3.reeple.ai/bill-payment/history
```

```bash
curl -X GET https://api-v3.reeple.ai/bill-payment/history \
  -H "api-key: YOUR_SECRET_KEY"
```
