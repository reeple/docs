---
title: List TV packages
description: List available cable TV bundle packages for a provider
---

List the bundle packages available for a given cable TV provider, before purchase.

## Endpoint

```
GET https://api-v3.reeple.ai/bill-payment/tv-services/{serviceCategoryId}
```

| Parameter | Type | Description |
|---|---|---|
| `serviceCategoryId` | String | **Required.** The ID of the cable provider whose packages you want — see [TV services](/api-reference/bill-payment/tv-services). |
