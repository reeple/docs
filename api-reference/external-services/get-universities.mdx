title: Get universities
description: List universities that accept tuition payments in a given country
---

List the universities you can pay tuition to in a given country.

## Endpoint

```
GET https://api-v3.reeple.ai/tuition/{country}/universities
```

| Parameter | Type | Description |
|---|---|---|
| `country` | String | **Required.** The country to list universities for. |

See also [Universities](/api-reference/tuition/universities) and [Create tuition](/api-reference/tuition/create-tuition).
