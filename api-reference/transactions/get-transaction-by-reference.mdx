Retrieve a single transaction by its reference.

## Endpoint

```
GET https://api-v3.reeple.ai/transactions/{reference}
```

| Parameter | Type | Description |
|---|---|---|
| `reference` | String | **Required.** Unique reference code identifying the transaction. |
