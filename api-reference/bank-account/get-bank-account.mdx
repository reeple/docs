title: Get a bank account
description: Retrieve details for a single bank account
---

Retrieve details for a single bank account by ID.

## Endpoint

```
GET https://api-v3.reeple.ai/accounts/{id}
```

| Parameter | Type | Description |
|---|---|---|
| `id` | String | **Required.** The unique identifier of the account you want to retrieve. |

## Response

{
  "success": true,
  "statusCode": 200,
  "message": "Fetched bank account",

</Note>
